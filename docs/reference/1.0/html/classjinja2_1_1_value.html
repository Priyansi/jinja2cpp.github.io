Detailed Description

Generic value class.

Variant-based class which is used for passing values to and from Jinja2C++ template engine. This class store the following types of values:

  • EmptyValue. In this case instance of this class threated as 'empty'
  • Boolean value.
  • String value.
  • Wide string value
  • String view value (nonstd::string_view)
  • Wide string view value (nonstd::wstring_view)
  • integer (int64_t) value
  • floating point (double) value
  • Simple list of other values (ValuesList)
  • Simple map of other values (ValuesMap)
  • Generic list of other values (GenericList)
  • Generic map of other values (GenericMap)
  • User-defined callable (UserCallable)

Exact value can be accessed via nonstd::visit method applied to the result of the Value::data() call or any of asXXX method (ex. Value::asString). In case of string retrieval it's better to use AsString or AsWString functions. Thay hide all nececcary transformations between various types of strings (or string views).
