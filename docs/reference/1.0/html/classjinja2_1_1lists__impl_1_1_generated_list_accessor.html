See implementation notes for the interface. This method should return unique pointer (with custom deleter) to the ListEnumerator interface implementation. Enumerator implementation should follow the requirements for the particular list type implementation.

nonstd::optional<size_t> jinja2::lists_impl::GeneratedListAccessor::GetSize ( ) const
inlineoverridevirtual

Called to get size of the list if applicable.

See implementation notes for the interface. This method should return valid (non-empty) size only for random-access sequences. In other cases this method should return empty optional.

Method can be called several times from the different threads.

Returns
Non-empty optional with the valid size of the list or empty optional in case of non-random sequence implementation
