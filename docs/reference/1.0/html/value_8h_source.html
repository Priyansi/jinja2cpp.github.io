521 /*!
522  * \brief User-callable descriptor
523  *
524  * This descriptor is used for description of the user-defined callables passed to the Jinja2C++ engine. Information
525  * from this descriptor is used by the engine to properly parse and prepare of the call parameters and pass it to the
526  * user-callable. For instance, such kind of user-defined callable passed as a parameter:
527  * ```c++
528  * jinja2::UserCallable uc;
529  * uc.callable = [](auto& params)->jinja2::Value {
530  * auto str1 = params["str1"];
531  * auto str2 = params["str2"];
532  *
533  * if (str1.isString())
534  * return str1.asString() + " " + str2.asString();
535  *
536  * return str1.asWString() + L" " + str2.asWString();
537  * };
538  * uc.argsInfo = {{"str1", true}, {"str2", true}};
539  * params["test"] = std::move(uc);
540  * ```
541  * This declaration defines user-defined callable which takes two named parameters: `str1` and `str2`. Further, it's
542  * possible to call this user-defined callable from the Jinja2 template this way:
543  * ```jinja2
544  * {{ test('Hello', 'World!') }}
545  * ```
546  * or:
547  * ```
548  * {{ test(str2='World!', str1='Hello') }}
549  * ```
550  * Jinja2C++ engine maps actual call parameters according the information from \ref UserCallable::argsInfo field and
551  * pass them as a \ref UserCallableParams structure. Every named param (explicitly defined in the call or it's default value)
552  * passed throught \ref UserCallableParams::args field. Every extra positional param mentoined in call passed as \ref
553  * UserCallableParams::extraPosArgs. Every extra named param mentoined in call passed as \ref
554  * UserCallableParams::extraKwArgs.
555  *
556  * If any of argument, marked as `mandatory` in the \ref UserCallable::argsInfo field is missed in the point of the
557  * user-defined call the call is failed.
558  */
559 struct UserCallable
560 {
561  //! Functional object which is actually handle the call
562  std::function<Value (const UserCallableParams&)> callable;
563  //! Information about arguments of the user-defined callable
564  std::vector<ArgInfo> argsInfo;
565 };
