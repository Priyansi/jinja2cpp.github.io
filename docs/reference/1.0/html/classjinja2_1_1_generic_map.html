Detailed Description

Helper class for accessing maps specified by the MapItemAccessor interface.

In the Value type can be stored either ValuesMap instance or GenericMap instance. ValuesMap is a simple dictionary object based on std::unordered_map. Rather than GenericMap is a more robust object which can provide access to the different types of dictionary entities. GenericMap takes the MapItemAccessor interface instance and uses it to access particular items in the dictionaries.
